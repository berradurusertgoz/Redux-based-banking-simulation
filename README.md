# Redux Mini Banking App

A small banking simulation built with React and Redux to practice global state management and async logic.

## Features
- Deposit & withdraw operations
- Loan request and repayment logic
- Centralized state management with Redux
- Clear separation of reducers and actions
- Scalable folder structure

## Tech Stack
- React
- Redux Toolkit
- JavaScript
- CSS

## Purpose
This project was created to strengthen my understanding of Redux concepts such as reducers, actions, store configuration, and predictable state updates.
